=== ВСЕ КОМАНДЫ ДЛЯ ВЫПОЛНЕНИЯ ЗАДАНИЙ ===

1. Запуск Docker:
   docker-compose up -d

2. Выполнение SQL заданий:
   docker exec todo-postgres psql -U todo_user -d todo_db -f /docker-entrypoint-initdb.d/01-create-tables.sql
   docker exec todo-postgres psql -U todo_user -d todo_db -f /docker-entrypoint-initdb.d/02-crud-operations.sql
   ... и т.д. для всех файлов sql/

3. Проверка базы данных:
   docker exec todo-postgres psql -U todo_user -d todo_db -c "\dt"
   docker exec todo-postgres psql -U todo_user -d todo_db -c "SELECT * FROM users;"
   docker exec todo-postgres psql -U todo_user -d todo_db -c "SELECT * FROM todos;"

4. Запуск ORM демо:
   cd src
   node orm-demo.js

5. Redis команды:
   docker exec todo-redis redis-cli ping
   docker exec todo-redis redis-cli SET "demo" "value"
   docker exec todo-redis redis-cli GET "demo"

6. PgAdmin:
   Открыть http://localhost:5050
   Логин: admin@todo.com
   Пароль: admin

7. Остановка всего:
   docker-compose down

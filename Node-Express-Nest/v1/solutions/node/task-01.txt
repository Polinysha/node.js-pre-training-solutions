Task 01: Custom Event Emitter - MessageSystem

Описание реализации:

1. Класс MessageSystem:
- Наследуется от EventEmitter
- Реализует pub-sub систему для сообщений
- Управляет пользователями и историей сообщений

2. Функциональность:
- Отправка сообщений разных типов: message, notification, alert, user-joined, user-left
- Подписка на все сообщения или конкретные типы
- Управление пользователями: добавление, удаление, получение списка
- Ведение истории сообщений (последние 10)
- Сбор статистики по типам сообщений

3. Основные методы:
- sendMessage(type, content, sender): отправка сообщения
- subscribeToMessages(callback): подписка на все сообщения
- subscribeToType(type, callback): подписка на конкретный тип
- addUser(username): добавление пользователя
- removeUser(username): удаление пользователя
- getActiveUsers(): получение списка активных пользователей
- getMessageHistory(): получение истории сообщений
- getStats(): получение статистики

4. Формат сообщения:
{
  id: уникальный идентификатор,
  type: тип сообщения,
  content: содержимое,
  timestamp: время отправки,
  sender: отправитель
}

5. Статистика:
- Общее количество сообщений
- Количество активных пользователей
- Размер истории сообщений
- Количество сообщений по типам

6. Использование:
- Система уведомлений
- Чат-приложения
- Мониторинг событий
- Логирование действий

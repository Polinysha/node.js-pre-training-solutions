Task 04: Mini HTTP Server for ToDos

Описание реализации:
- Реализован RESTful API сервер для управления задачами (ToDos)
- Использованы встроенные модули Node.js: http и url
- Поддержка всех CRUD операций: GET, POST, PUT, DELETE
- Валидация данных согласно требованиям задания
- Консистентный формат JSON ответов
- Поддержка CORS для работы с веб-клиентами
- Фильтрация задач по параметру completed
- Обработка ошибок с соответствующими HTTP статусами

Как запустить решение:
1. Убедитесь, что Node.js установлен
2. Перейдите в директорию с файлом task-04.js
3. Запустите сервер: node task-04.js
4. Сервер будет доступен по адресу: http://localhost:3000

Доступные эндпоинты:
- GET    /todos               - Получить все задачи
- POST   /todos               - Создать новую задачу
- GET    /todos/:id           - Получить задачу по ID
- PUT    /todos/:id           - Обновить задачу по ID
- DELETE /todos/:id           - Удалить задачу по ID
- GET    /todos?completed=true - Отфильтровать задачи по статусу

Примеры использования:
- Создание задачи:
  POST /todos
  Body: {"title": "Новая задача", "description": "Описание"}

- Получение всех задач:
  GET /todos

- Обновление задачи:
  PUT /todos/1
  Body: {"title": "Обновленная задача", "completed": true}

- Фильтрация завершенных задач:
  GET /todos?completed=true

Файлы реализации:
- task-04.js     - Основной файл с реализацией сервера
- task-04-test.js - Тесты для проверки функциональности
